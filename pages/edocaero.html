Description
Edocaero a été le premier projet de stage que j'ai réalisé, il a été très compliqué à installer sur un environnement Windows car nous n'avions pas les droits administrateurs. L'installation et configuration a prit plusieurs jours pendant lesquels nous avons profité pour apprendre les bases de React notamment via les cours de notre chef de projet mais aussi grâce à Open Classroom. En effet, n'ayant jamais cottoyé Node et React, c'était assez compliqué de s'acclimater à ce changement de langage. Une fois l'application prise en main, j'ai dû réaliser un CRUD (Create, Read, Update, Delete) de la page Accessibilité de cette application. C'était assez compliqué au début mais cela s'est facilité avec le temps. J'avais ensuite à faire le Frontend avec un tableau regroupant les informations et le remplir via une API REST.
