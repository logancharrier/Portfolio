Description
Open Notebook AI est un projet que j’ai développé pour explorer l'auto-hébergement d'intelligences artificielles, avec une architecture entièrement conteneurisée sous Docker. L'idée était de créer un environnement permettant d'exécuter localement des modèles de langage et d'embedding, sans dépendre d’un service cloud externe. Dès le début, le défi principal était de faire interagir trois services clés :

- Open Notebook AI (port 8080) : Interface web pour interagir avec l’IA.
- Ollama (port 11434) : Serveur hébergeant le modèle Llama 3.2.
- SurrealDB (port 8000) : Base de données optimisée pour stocker les embeddings et gérer les requêtes complexes.

Ce projet a été une vraie plongée dans l’orchestration de conteneurs. J’ai dû non seulement configurer et relier ces services, mais aussi comprendre comment optimiser la communication entre eux. Un des moments clés a été la mise en place de SurrealDB, qui m’a obligé à revoir ma gestion des données pour exploiter efficacement les embeddings. Un autre défi a été l’intégration du modèle avec Ollama. J'ai dû creuser la documentation, expérimenter avec différentes versions du modèle et optimiser son chargement pour éviter des ralentissements. Ce projet m'a permis de renforcer mes compétences en développement backend, orchestration de conteneurs et intelligence artificielle open-source. Travailler sur une IA locale et auto-hébergée a aussi été un bon exercice pour comprendre les enjeux de performance et d’indépendance technologique.
