Description
Ce projet a été réalisé lors de mon stage de deuxième année de BTS SIO afin de préparer mon épreuve E5. J'ai décidé de réaliser quelque chose en HTML CSS simple pour pouvoir être libre de faire chaque chose comme je l'imaginais. J'avais d'abord une première version de portfolio que j'avais réalisé sur un CMS en première année mais malheureusement, je n'aimais pas du tout le design et le rendu final. Pour cette nouvelle version, j’ai versionné l’ensemble du code sur GitHub et hébergé le site via GitHub Pages, configuré sur le nom de domaine logancharrier.fr grâce à OVH. Désormais, c'est un portfolio qui me satisfait et qui est à mon image.
